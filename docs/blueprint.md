# **App Name**: Aura AI

## Core Features:

- Interactive Chat Interface: Engage with the AI through a responsive and intuitive chat interface to input queries and receive AI-generated text responses.
- Knowledge Base Query Tool: Utilize an AI tool to efficiently search and retrieve information from an integrated knowledge base to provide informed answers to user questions.
- Content Summarization: The AI can summarize longer pieces of text, articles, or past conversations into concise, digestible points.
- Conversation History: Access and review a chronological history of past interactions with the AI for continuity and reference.
- Personalized Preferences: Allow users to store and apply preferences such as AI tone or response verbosity, affecting future interactions.
- Next.js AI Gateway: Leverage Next.js API routes to securely handle and route user requests to various generative AI models.

## Style Guidelines:

- Primary color: Muted deep violet (#6B66CC) to evoke a sense of calm intelligence and sophistication.
- Background color: Dark desaturated purple (#24202D), providing a professional, eye-friendly base for extended use in a dark scheme.
- Accent color: Light, vibrant sky blue (#A3DBFF) to highlight interactive elements and draw attention.
- Headline and body font: 'Inter' (sans-serif) for its modern, objective, and highly legible appearance across all text.
- Code font: 'Source Code Pro' (monospace sans-serif) for displaying any AI-generated code snippets or technical output.
- Use minimalist, line-based icons to maintain a clean, modern aesthetic that complements the dark theme and technical nature of the app.
- Implement a clean and spacious layout with clear content hierarchy, ensuring ease of reading and intuitive navigation through AI interactions.
- Incorporate subtle transition animations for chat bubble appearance, loading states, and content updates to enhance the user experience without distraction.
